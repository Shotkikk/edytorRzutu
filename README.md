# E X L E C T

Exlect is a pure way to make your home wiring plane.

  - Select plane
  - CropIt!
  - Add some components...
  - Done!!!

### Installation

Exlect requires gulp to run.

Build whole app to dist folder:

```sh
$ gulp build
```

Watching app folder and reloading server:

```sh
$ gulp watch
```
